# 🍽️ Responsive Restaurant Landing Page

A modern, elegant, and fully responsive restaurant landing page built using **HTML5** and **CSS3**. This project was developed as **Task 2** during my **Web Development Internship at Synent Technologies**.

The website focuses on clean UI, responsive layouts, and an engaging user experience across desktop, tablet, and mobile devices.
