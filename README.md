# 🍽️ DuDu Kitchen — Responsive Restaurant Landing Page

A responsive restaurant landing page built with **HTML & CSS only** as part of the **Synent Internship — Task 2**.

---

## 📸 Preview

> An elegant, luxury-themed single-page landing page for *DuDu Kitchen* — an Ethiopian fine dining restaurant in Bishoftu, Ethiopia.

---

## 📋 Sections

| Section | Description |
|---|---|
| **Hero** | Full-screen welcome with headline, CTA buttons, and key stats |
| **Features** | 4 cards highlighting what makes DuDu Kitchen special |
| **Menu** | Seasonal chef's picks with dish names, descriptions & prices |
| **Footer** | Brand info, navigation links, contact details, and socials |

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties, Media Queries)
- Google Fonts (Lora & Nunito)

---

## ✨ Features

- Elegant deep green, gold & cream color palette
- Fully responsive layout (mobile, tablet, desktop)
- Fixed navigation bar with smooth scroll
- Hover animations on cards and buttons
- Gold ornament dividers for a luxury feel
- Mobile-friendly collapsed navigation

---

## 🚀 Getting Started

No installation needed. Just open the file in your browser:

```bash
# Clone the repository
git clone https://github.com/DuDu21cs/Synent-task2-ResponsiveLandingPage-DuresaChemeda.git

# Open in browser
cd Synent-task2-ResponsiveLandingPage-DuresaChemeda
open task2-landing-page.html
```

## 📱 Responsive Breakpoints

| Screen Size | Layout |
|---|---|
| Desktop (> 768px) | 2-column features, 3-column menu |
| Tablet (≤ 768px) | 1-column features, 2-column menu |
| Mobile (≤ 480px) | Everything stacks to single column |
